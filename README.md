TP - Análise de Protocolos de Aplicação e Transporte

Este repositório contém o projeto prático de construção, implementação e análise de uma topologia de redes, desenvolvido no simulador Cisco Packet Tracer.

Foi construída e analisada uma topologia em que cinco redes distintas se comunicam por meio de roteadores interligados por enlaces com diferentes capacidades. O foco principal é observar como serviços de aplicação usam os protocolos de transporte UDP e TCP e como a topologia afeta o caminho percorrido pelos dados.
